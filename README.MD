## Описание проекта
Учебный программный проект, предназначенный для сбора данных с веб-ресурсов.
Программа выполняет загрузку HTML-страниц сайтов, извлекает необходимые данные
и сохраняет их в структурированном формате.

## Технологии
- Python
- requests
- BeautifulSoup4
- pandas
- Git
